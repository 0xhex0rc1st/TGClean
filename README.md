# TGClean

A CLI tool for Telegram that deletes your sent messages, clears private chat histories, leaves groups and channels, and deletes groups/channels you own using the Telegram API (Telethon).

## Features
* Delete messages sent from your account
* Clear private chat histories
* Leave groups and channels
* Delete groups and channels you own
* Supports Telegram 2FA
* Interactive terminal interface
## Installation
```
git clone https://github.com/0xhex0rc1st/TGClean.git
cd TGClean
pip install -r requirements.txt
```
## Setup

Get your Telegram API credentials from:

https://my.telegram.org → API Development Tools

Run the script and enter your API ID and API Hash when prompted.
